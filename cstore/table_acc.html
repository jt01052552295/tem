주의사항

  • col width 대신 col style="width" 권장. HTML5 validation에서 오류로 체크됨
  • mBoard 의 th width는 16%~20%로 컨텐츠 분량에 따라, 자유롭게 사용하며 분량이 많아질 경우엔 적당한 단어에서 br을 삽입해서 정렬을 맞춘다.
  • 체크박스만 셀에 삽입될경우 .chk로 width값 제어
  • 셀병합될경우 tr에 .rowMerge 추가하여 보더제어(기본은 tr 첫번째 td에 왼쪽 보더없음)
  • .eTr : tr 오버시 배경색 변경에 사용 (행을 선택할 경우만)
  • 게시판 리스트일 경우 '테이터 없음'셀 하단에 추가
  • 폼요소의 부가 설명은 .txtInfo 사용
  • td 폭을 좁게 사용해야 하는 경우 tbody에 .gCellSingle 추가 : 좌우 패딩 5px로 변경
  • 셀 내에서 버튼 정렬은 [button] 페이지를 참고한다
